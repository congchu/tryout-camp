# logs/

작업 과정의 로그와 기록을 저장하는 폴더.

## 파일 설명

| 파일 | 용도 |
|------|------|
| `WORKLOG.md` | 날것의 작업 히스토리 로그 (날짜별 기록) |
| `worklog/` | 주제별 상세 토론/분석 기록 |
| `EDITOR_SKILL.md` | 편집자 스킬 가이드 |
| `workbook-draft.md` | 워크북 초안 |
| `troubleshooting-*.md` | 문제 해결 기록 |

## 규칙

- 작업 중 발생한 시행착오, 의사결정, 토론 내용 기록
- 정리된 문서가 아닌 **날것의 로그** 유지
- 나중에 "왜 이렇게 했지?" 파악할 수 있게 작성
