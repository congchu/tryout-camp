# Designer Portfolio Style Guide

4가지 타입의 디자이너 포트폴리오 스타일 가이드

---

## Type A: Minimal + Bold Typography

> 참고: Jordan Gilroy (jordangilroy.com)

### 컨셉
- 극도로 미니멀한 레이아웃
- 거대한 타이포그래피가 주인공
- 여백이 디자인 요소

### 색상 시스템
| 용도 | 색상 | HEX |
|------|------|-----|
| 배경 | 화이트 | #FFFFFF |
| 텍스트 | 블랙 | #000000 |
| 서브텍스트 | 그레이 | #666666 |
| 액센트 | 없음 (텍스트로 강조) | - |

### 타이포그래피
| 요소 | 크기 | 굵기 | 서체 |
|------|------|------|------|
| Hero Title | 120px+ | 900 (Black) | Monument Extended, Bebas Neue |
| Section Title | 72px | 700 | 동일 |
| Body | 18px | 400 | Inter, Helvetica Neue |
| Caption | 14px | 400 | 동일 |

### 레이아웃
- 단일 컬럼 중심
- 수평 스크롤 섹션 (선택)
- 섹션 간 간격: 200px+
- 좌우 패딩: 48-80px

### 특징적 요소
- 시계/날짜 위젯
- 마우스 따라다니는 커스텀 커서
- 텍스트만으로 구성된 네비게이션
- 프로젝트 번호 표기 (01, 02, 03...)

### 인터랙션
- 호버 시 텍스트 색상 반전
- 부드러운 스크롤 (scroll-behavior: smooth)
- 텍스트 reveal 애니메이션

### 금지 사항
- 복잡한 그래픽 요소
- 3개 이상의 색상
- 둥근 모서리 (border-radius)
- 그림자 효과

## Type B: Dark Theme + Grid

> 참고: Adrian Zumbrunnen (azumbrunnen.me)

### 컨셉
- 다크 배경으로 작업물 강조
- 정교한 그리드 시스템
- 마이크로 인터랙션 풍부

### 색상 시스템
| 용도 | 색상 | HEX |
|------|------|-----|
| 배경 | 다크 | #0A0A0A |
| 서피스 | 다크 그레이 | #141414 |
| 텍스트 | 화이트 | #FFFFFF |
| 서브텍스트 | 그레이 | #888888 |
| 액센트 | 퍼플/블루 | #6366F1 |
| 보조 액센트 | 핑크/그린 | #EC4899 / #10B981 |

### 타이포그래피
| 요소 | 크기 | 굵기 | 서체 |
|------|------|------|------|
| Hero Title | 64px | 600 | Inter, SF Pro |
| Section Title | 48px | 600 | 동일 |
| Body | 16px | 400 | 동일 |
| Code/Tag | 14px | 500 | JetBrains Mono |

### 레이아웃
- 12컬럼 그리드 시스템
- 카드 기반 프로젝트 배치
- 섹션 간 간격: 120px
- 컨테이너 최대 너비: 1200px

### 특징적 요소
- 커스텀 커서 (원형, 색상 변화)
- 프로젝트 카드 호버 시 확대
- 색상 팔레트 표시 (케이스 스터디)
- 그리드 라인 시각화 (배경)

### 인터랙션
- 커서 블렌드 모드 (mix-blend-mode: difference)
- 카드 3D tilt 효과
- 스크롤 기반 패럴랙스
- 페이지 전환 애니메이션

### 금지 사항
- 밝은 배경색
- 정렬되지 않은 요소
- 과도한 그라데이션
- 저해상도 이미지

## Type C: Light & Clean

> 참고: Rachel How (rachelhow.com)

### 컨셉
- 밝고 친근한 분위기
- 글래스모피즘/소프트 UI
- 개인 브랜딩 강조

### 색상 시스템
| 용도 | 색상 | HEX |
|------|------|-----|
| 배경 | 오프화이트 | #FAFAFA |
| 서피스 | 화이트 | #FFFFFF |
| 텍스트 | 다크 그레이 | #1A1A1A |
| 서브텍스트 | 미디엄 그레이 | #6B7280 |
| 액센트 | 소프트 퍼플 | #8B5CF6 |
| 그라데이션 시작 | 핑크 | #F472B6 |
| 그라데이션 끝 | 퍼플 | #8B5CF6 |

### 타이포그래피
| 요소 | 크기 | 굵기 | 서체 |
|------|------|------|------|
| Hero Title | 56px | 700 | Poppins, Plus Jakarta Sans |
| Section Title | 36px | 600 | 동일 |
| Body | 16px | 400 | Inter |
| Button | 14px | 500 | 동일 |

### 레이아웃
- 중앙 정렬 컨텐츠
- 카드형 프로젝트 (둥근 모서리)
- 섹션 간 간격: 100px
- 컨테이너 최대 너비: 1000px

### 특징적 요소
- Frosted glass 네비게이션 (backdrop-filter: blur)
- 그라데이션 텍스트/버튼
- 소프트 그림자 (box-shadow)
- 아바타/프로필 사진
- 뉴스레터 구독 섹션
- 리소스/블로그 링크

### 인터랙션
- 버튼 호버 시 그라데이션 이동
- 카드 호버 시 살짝 떠오름 (translateY)
- 부드러운 fade-in 애니메이션
- 스크롤 시 네비게이션 배경 변화

### 금지 사항
- 날카로운 모서리
- 강한 대비 (순수 블랙)
- 복잡한 애니메이션
- 어두운 색상

## Type D: Bold Interaction

> 참고: Constance S (constancesouville.com)

### 컨셉
- 대담하고 실험적
- 인터랙션이 핵심 경험
- 규칙을 깨는 레이아웃

### 색상 시스템
| 용도 | 색상 | HEX |
|------|------|-----|
| 배경 | 크림/오프화이트 | #F5F5F0 |
| 텍스트 | 블랙 | #000000 |
| 액센트 1 | 레드 | #FF3333 |
| 액센트 2 | 블루 | #0066FF |
| 액센트 3 | 옐로우 | #FFCC00 |

### 타이포그래피
| 요소 | 크기 | 굵기 | 서체 |
|------|------|------|------|
| Hero Title | 80-150px | 900 | Clash Display, Space Grotesk |
| Section Title | 60px | 700 | 동일 |
| Body | 20px | 400 | 동일 |
| Menu | 48px+ | 700 | 동일 |

### 레이아웃
- 비대칭 그리드
- 오버사이즈 요소
- 섹션 간 간격: 불규칙 (80-200px)
- 전체 화면 활용 (패딩 최소화)

### 특징적 요소
- 거대한 커스텀 커서 (80px+)
- 오버사이즈 버튼
- 텍스트 기반 풀스크린 메뉴
- 프로젝트 이미지 호버 시 등장
- 숫자/인덱스 강조
- 실제 라이브 사이트 링크

### 인터랙션
- 마우스 따라다니는 이미지
- 커서 크기 변화 (호버 시 확대)
- 텍스트 스크램블/글리치 효과
- 드래그 스크롤
- 페이지 전환 와이프 효과

### 금지 사항
- 조용하고 미니멀한 접근
- 작은 인터랙션
- 일반적인 레이아웃
- 안전한 색상 조합

## 공통 기술 요구사항

### 반응형 브레이크포인트
```
Mobile: ~767px
Tablet: 768px ~ 1023px
Desktop: 1024px+
Large: 1440px+
```

### 필수 섹션 (모든 타입)
1. Hero (이름 + 직함)
2. About (소개)
3. Projects (작업물 3-6개)
4. Contact (연락처)

### 성능 최적화
- 이미지: WebP 포맷, lazy loading
- 폰트: font-display: swap
- CSS: critical CSS 인라인

### 접근성
- 색상 대비 최소 4.5:1
- 키보드 네비게이션 지원
- alt 텍스트 필수
- focus 상태 표시

## 프롬프트 작성 시 체크리스트

각 타입별 프롬프트 작성 시 아래 항목을 모두 명시:

- [ ] 색상 HEX 값 전체
- [ ] 폰트 이름, 크기, 굵기
- [ ] 각 섹션 구조와 순서
- [ ] 간격 수치 (px 단위)
- [ ] 인터랙션 상세 (duration, easing)
- [ ] 금지 사항
- [ ] 출력 형식 (단일 HTML 등)
