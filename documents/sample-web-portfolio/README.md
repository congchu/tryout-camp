# 샘플 포트폴리오 프로젝트

## 배경

실험마켓 첫 프로젝트 **"AI로 포트폴리오 만들기 5일 챌린지"** (크크웍스 대상)
→ 신청 페이지에 보여줄 샘플 포트폴리오 제작

## 목표

1. 직군별 샘플 포트폴리오 3개 제작
2. 프롬프트를 "동일 결과 재현 가능" 수준으로 고도화
3. 시행착오 과정을 강의자료/성장기로 기록

## 포트폴리오 제작 4단계

> 이 과정이 5일 챌린지 커리큘럼이 됨

```
Step 1. 브랜딩 (WHY) → 어떤 사람으로 보여지고 싶은가?
Step 2. 구조 (WHAT) → 그걸 전달하려면 뭘 보여줘야 하나?
Step 3. UI/UX (HOW) → 어떤 스타일로 표현할 것인가?
Step 4. 프롬프트 → 재현 가능하게 고정
```
