# Day 3: 콘텐츠 완성하기

## 핵심 메시지
Day 1-2는 "멋진 사이트 따라 만들기"였다면, Day 3부터는 **진짜 내 포트폴리오**로 만드는 단계.
멋지게만 만들면 안 됨 → **목적에 맞아야 함**.

> 포트폴리오는 "설득하는 글"이다.
> - 주장이 있어야 함
> - 설득이 있어야 함
> - 증거 있으면 보여주고, 없으면 설득

---

## 4단계 흐름

### Step 1. 목적 정하기 (포트폴리오론)
- **왜 만드는지**: 취업? 프리랜서 수주? 개인 브랜딩?
- **누가 볼 건지**: 채용담당자? 클라이언트? 일반 대중?
- **그 사람이 뭘 기대하는지**: 실력 증명? 신뢰감? 스타일?
- → **그래서 뭘 보여줘야 하는지** 결정

### Step 2. 자료 올리기
- Codex에 **있는 자료 1~3개만** 올리기 (이력서, 링크드인, 프로젝트 파일 등)
- 없으면 링크드인 URL 하나도 OK
- 완벽하게 준비 X → 있는 것부터 시작

### Step 3. 카피 뽑기
- 올린 자료 기반으로 AI한테 글 요청
- Step 1에서 정한 **목적에 맞게** 카피라이팅 가공
- 단순 정보 나열 X → 설득력 있는 문장으로

### Step 4. 웹사이트에 적용
- Day 2에서 만든 결과물에 콘텐츠 채우기
- 텍스트 + 이미지 교체

---

## 예상 어려움 + 대응 방법

### Step 1. 목적 정하기
| 어려움 | 대응 |
|--------|------|
| **"그냥 멋있어 보이고 싶어요"** | 구체적 질문으로 유도: "누가 이 포트폴리오를 보고 연락을 줬으면 좋겠어요?" |
| **타겟 모호** | 예시 제시: "디자인 에이전시 대표" vs "스타트업 채용담당자" 차이 설명 |
| **"나는 특별한 게 없는데..."** | 차별점 ≠ 대단한 것. "이 사람한테 맡기면 OO하겠다"만 있으면 됨 |

### Step 2. 자료 올리기
| 어려움 | 대응 |
|--------|------|
| **"자료가 없어요"** | 링크드인 URL, 카톡 자기소개, 인스타 바이오라도 OK |
| **"뭘 올려야 하지?"** | 1~3개만. 완벽하게 준비 X, 있는 것부터 |
| **프로젝트 기록 없음** | 결과물 스크린샷만 있어도 OK |

### Step 3. 인터뷰/카피라이팅
| 어려움 | 대응 |
|--------|------|
| **질문에 답 못함** | 가벼운 질문부터 시작. "요즘 뭐 해요?" → "그거 왜 해요?" 자연스럽게 |
| **AI 결과가 어색** | "내 말투로 다시" / "더 캐주얼하게" 수정 요청. 초안일 뿐 |
| **자기 PR 어려움** | 제3자 시점으로 쓰기. "이 사람은 ~한 사람입니다" |

### Step 4. 웹사이트 적용
| 어려움 | 대응 |
|--------|------|
| **구조 매칭 어려움** | 템플릿 섹션별로 "여기에 뭐 넣을지" 미리 매핑 제공 |
| **이미지 없음** | 스크린샷도 OK, Canva 목업도 OK. 완벽한 이미지 아니어도 됨 |
| **텍스트 길이 안 맞음** | AI에게 "3줄로 줄여줘" / "좀 더 늘려줘" 조절 |

## FAQ (예상 질문)

### 자료 관련
**Q. 진짜 자료가 하나도 없어요**
→ 이력서도 없으면 링크드인, 카톡 자기소개, 인스타 바이오라도. 0에서 시작하는 사람 없음.

**Q. 민감한 정보(고객사명, 금액 등)는 어떻게 해요?**
→ 가명 처리하거나 "대기업 A사" 식으로. AI가 알아서 익명화해줌.

**Q. 예전 작업물인데 퀄리티가 별로예요**
→ 성장 스토리로 활용. "이때는 이랬는데, 지금은 이렇게 발전했다"

### 카피라이팅 관련
**Q. AI가 뽑은 문장이 너무 과장돼요**
→ "더 담백하게" / "사실 기반으로만" 요청. 톤 조절 가능.

**Q. 영어로 써야 하나요?**
→ 타겟에 따라 다름. 국내 클라이언트면 한글, 해외면 영어. 섞어도 됨.

**Q. 소개글이 너무 길어요/짧아요**
→ AI에게 글자수 지정 요청. "100자 이내로" / "3문장으로"

### 적용 관련
**Q. Day 2 템플릿 구조가 내 콘텐츠랑 안 맞아요**
→ 섹션 순서 바꾸거나 삭제 가능. "About 섹션 삭제해줘" 요청하면 됨.

**Q. 이미지 사이즈가 안 맞아요**
→ Codex에 "이미지 비율 맞춰줘" 요청. 또는 Canva에서 리사이즈.
