# Day 5: 사람들이 봤을까?

## 핵심 메시지
포트폴리오 만들었으면 끝? **아니다.**
몇 명이 봤는지, 어디까지 봤는지 알아야 개선할 수 있다.

> 내 포트폴리오, 누가 봤는지 확인하는 방법!

---

## 3단계 흐름

### Step 1. 왜 확인해야 할까?
- 링크 보냈는데 봤는지 안 봤는지 모름
- "봤어요~" 말만 하고 실제로 안 봤을 수도
- 어디서 이탈했는지 알면 → 개선 가능

### Step 2. 방문자 확인 설정하기
- Mixpanel 연동 (Codex가 해줌)
- 설치하면 자동으로 방문자 수 집계
- 결과: 대시보드에서 확인 가능

### Step 3. 챌린지 마무리
- 5일간 만든 것 정리
- 앞으로 활용법
- 다음 단계 안내

---

## 예상 어려움 + 대응 방법

### Step 1. 왜 확인해야 할까?
| 어려움 | 대응 |
|--------|------|
| **"그냥 물어보면 되지 않나?"** | 직접 물어보기 어색함 + 거짓말 가능 |
| **"나는 그냥 있으면 되는데"** | 나중에 개선하려면 필요함 |

### Step 2. 방문자 확인 설정하기
| 어려움 | 대응 |
|--------|------|
| **Mixpanel이 뭔지 모름** | "방문자 수 세어주는 서비스"라고만 설명 |
| **영어 사이트라 어려움** | Codex가 설정 다 해줌 |
| **뭘 확인해야 하는지 모름** | "몇 명이 봤는지"만 우선 |

### Step 3. 챌린지 마무리
| 어려움 | 대응 |
|--------|------|
| **아직 부족한 느낌** | 정상임. 이건 "시작"이었다. |
| **뭘 더 해야 할지 모름** | 다음 단계 안내 제공 |

---

## FAQ (예상 질문)

### 방문자 확인 관련
**Q. 몇 명이 봤는지만 알면 되는 거 아니에요?**
→ 맞아요. 일단은 그것만 확인해도 충분!

**Q. 무료예요?**
→ 네. Mixpanel 무료 티어로 충분합니다.

**Q. 방문자 정보가 어디까지 보여요?**
→ 몇 명, 언제, 어느 나라에서 봤는지 정도. 개인정보는 안 보임.

**Q. 설치 후 바로 보여요?**
→ 설치 후 방문이 있어야 데이터가 쌓임. 본인이 접속해보면 바로 확인 가능.

### 챌린지 마무리 관련
**Q. 5일 했는데 아직 부족해요**
→ 당연합니다! 이건 "시작"이에요. 계속 다듬어가면 됩니다.

**Q. 앞으로 뭘 더 하면 좋을까요?**
→ 프로젝트 추가, 디자인 개선, 커스텀 도메인 등 (후속 과정 안내)

```

```


## 챌린지 완료 후 상태

5일 후, 수강생은:
- ✅ 실제 URL이 있는 웹 포트폴리오

- ✅ 카톡 공유 시 미리보기 뜸
- ✅ 몇 명이 봤는지 확인 가능
- ✅ 필요할 때 직접 수정 가능
