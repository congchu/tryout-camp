# 실험마켓 캠프 - 온보딩 가이드

이 문서를 읽으면 이 프로젝트가 뭔지, 왜 존재하는지, 어떻게 운영되는지 전부 이해할 수 있다.

---

## 1. 실험마켓이 뭐야?

### 한 줄 정의
**직장인을 위한 수익화 실험 플랫폼**

### 배경
직장인들 중에 "나도 뭔가 만들어서 팔아볼 수 있을까?"라는 생각을 하는 사람들이 많다. 전자책, 강의, 템플릿, 사이드 프로젝트 등. 근데 대부분 시작도 못 하거나, 시작해도 금방 포기한다.

왜? **너무 크게 시작하려고 해서**. 완벽한 상품, 완벽한 마케팅, 완벽한 준비.

실험마켓은 다르게 접근한다:
> "완벽하게 준비하지 말고, 작게 실험해봐. 9,900원이라도 벌어보면 그게 시작이야."

### 핵심 철학 (Manifesto)
- 완성된 답보다 **시작할 용기 있는 시도**를 존중한다
- 실패한 실험을 **숨기지 않고 기록**한다
- 판매보다 **실험을 먼저** 이야기한다
- 혼자가 아니라 **함께 실험하는 크루**

### 실험마켓에 있는 것들
| 상품 | 설명 |
|------|------|
| 수익화 코스 | 전자책 만들기, 크몽 수주 등 장기 코스 |
| 스킬 코스 | AI 도구, Framer 등 스킬 습득 |
| 커뮤니티 | 실험하는 사람들의 모임 |
| **캠프** | 단기 콜라보 챌린지 ← **이 프로젝트** |

## 2. 캠프가 뭐야?

### 한 줄 정의
**다른 브랜드와 콜라보하거나 특별한 주제로 진행하는 5-7일 단기 챌린지**

### 왜 캠프를 만들었어?
실험마켓의 철학이 "작게 실험"이잖아. 그걸 우리도 실천하는 거다.

- 새로운 주제를 **빠르게 테스트**할 수 있다
- 다른 브랜드와 **콜라보**하면 새로운 고객을 만날 수 있다
- 5-7일이면 **부담 없이** 참여할 수 있다
- 잘 되면 정규 코스로 확장하면 된다

캠프는 실험마켓의 **MVP이자 실험 공간**이다.

### 캠프의 특징
| 항목 | 내용 |
|------|------|
| 기간 | 5-7일 (짧게) |
| 형태 | 매일 미션 수행 |
| 결과물 | 실제로 뭔가 만들어냄 |
| 구조 | 전용 랜딩 페이지 + 워크북 |

### 캠프 예시
- **AI 포트폴리오 챌린지** (크크웍스 × 실험마켓): 비전공자가 AI로 6일 만에 포트폴리오 사이트 만들기
- (다음 캠프는 아직 기획 중)

## 3. 이 프로젝트(tryout-camp)는 뭐야?

### 한 줄 정의
**캠프 전용 웹 플랫폼**

### 왜 따로 분리했어?
실험마켓 메인 사이트(tryoutmarket.co)에 다 넣으면 너무 커진다. 캠프는 성격이 다르기도 하고:
- 캠프마다 다른 랜딩 페이지가 필요하다
- 빠르게 만들고 실험해야 한다
- 메인 사이트에 영향 주지 않고 독립적으로 운영하고 싶다

그래서 **별도 프로젝트로 분리**했다.

### 구조
```
tryoutmarket.co (메인 플랫폼)
├── 수익화 코스
├── 스킬 코스
└── 커뮤니티

tryout-camp (이 프로젝트)
└── 캠프들
    ├── AI 포트폴리오 (/cc/ai-portfolio)
    └── (다음 캠프)
```

## 5. 이 문서들은 뭐야?

### 왜 만들었어?
프로젝트를 하다 보면 이런 문제가 생긴다:
- "이거 왜 이렇게 했더라?"
- "지금 어디까지 온 거지?"
- "다음에 뭘 해야 하지?"

시간이 지나면 맥락이 끊긴다. 특히 혼자 하거나, Claude와 함께 하면 더 그렇다. 대화가 끊기면 맥락도 사라지니까.

그래서 **기준 문서**를 만들기로 했다. 나중에 다시 봐도 "아, 이래서 이랬구나" 이해할 수 있게.
