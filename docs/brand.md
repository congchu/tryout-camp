# 브랜드

이 문서는 실험마켓과 캠프의 정체성을 설명한다. "우리가 뭘 하는 곳인지" 이해하려면 이 문서를 읽으면 된다.

---

## 1. 실험마켓

### 왜 만들었어?

직장인들 사이에서 이런 고민이 많다:
- "나도 뭔가 만들어서 팔아볼 수 있을까?"
- "퇴근 후에 뭔가 해보고 싶은데, 뭘 해야 할지 모르겠어"
- "강의 들어봤는데, 결국 아무것도 안 했어"

문제는 **너무 크게 시작하려는 것**이다. 완벽한 상품, 완벽한 마케팅, 완벽한 준비. 그러다 지치고 포기한다.

실험마켓은 다르게 접근한다:
> "완벽하게 준비하지 말고, 작게 실험해봐."

### 핵심 철학

**"우리는 배움을 키트로, 시도를 경험으로 만듭니다."**

이 말이 뭔 뜻이냐면:
- 배움 → 그냥 지식이 아니라 **바로 써먹을 수 있는 키트**로 제공
- 시도 → 실패해도 괜찮은 **안전한 실험 경험**으로 만듦

### Manifesto (선언문)

우리가 믿는 것들:

| 선언 | 의미 |
|------|------|
| 완성된 답보다 **시작할 용기 있는 시도**를 존중한다 | 완벽할 필요 없어, 일단 해봐 |
| 수익화를 **모두가 실험할 수 있는 구조**로 바꾼다 | 특별한 사람만 하는 게 아니야 |
| 실패한 실험을 **숨기지 않고 기록**한다 | 실패도 가치 있어, 다음 사람에게 도움 됨 |
| 판매보다 **실험을 먼저** 이야기한다 | 팔기 전에 만들어보는 게 먼저 |
| 혼자가 아니라 **함께 실험하는 크루** | 같이 하면 덜 외롭고 더 멀리 감 |

### 타겟 사용자

실험마켓이 도와주려는 사람들:

**1. 사이드 프로젝트 실험가**
- 직장 다니면서 부업이나 창업 해보려는 사람
- "내 아이디어가 돈이 될까?" 빨리 검증하고 싶은 사람

**2. 프리랜서/1인 비즈니스**
- 이미 자기 서비스가 있는데, AI/자동화로 업그레이드하려는 사람
- 디자이너, 강사, 작가, 개발자 등

**3. 콘텐츠 수익화 입문자**
- 블로그, 유튜브, 전자책으로 첫 수익 내보고 싶은 사람
- 전 과정이 처음이라 "작게 시험"이 필요한 사람

**4. 커리어 확장가**
- 이직, 전환, 프리랜서 전향 준비하는 사람
- 포트폴리오나 자기 브랜딩을 실험하고 싶은 사람

### 공통점

이 사람들의 공통 심리:
- 완벽한 정규 과정보다 **짧고, 작게, 바로 해보는** 걸 선호
- 실패해도 부담 적은 실험 구조 원함
- **배움 + 결과물 + 실행 경험** 3박자 중요
- AI, 자동화, 최신 툴에 호기심
- 혼자보다 **작게라도 함께** 하고 싶음

## 2. 캠프

### 캠프가 뭐야?

실험마켓 안에 여러 상품이 있다:
- 수익화 코스 (장기, 깊이 있게)
- 스킬 코스 (특정 도구 습득)
- 커뮤니티 (상시 모임)
- **캠프** (단기, 이벤트성)

캠프는 그 중에서 **가장 짧고, 가장 실험적인** 형태다.

### 왜 캠프를 만들었어?

실험마켓의 철학이 "작게 실험"이잖아. 그걸 우리 스스로도 실천하는 거다.

**캠프의 역할:**
| 역할 | 설명 |
|------|------|
| MVP | 새로운 주제를 빠르게 테스트 |
| 유입 채널 | 콜라보로 새로운 고객 만남 |
| 실험 공간 | 잘 되면 정규 코스로 확장 |

### 캠프의 특징

일반 코스와 뭐가 달라?

| 구분 | 일반 코스 | 캠프 |
|------|-----------|------|
| 기간 | 4-8주 | 5-7일 |
| 형태 | 자기 주도 학습 | 매일 미션 수행 |
| 콘텐츠 | 영상 + 자료 | 워크북 (텍스트 중심) |
| 결과물 | 선택적 | 필수 (매일 미션) |
| 가격 | 유료 | 무료 또는 저가 |
| 목적 | 깊은 학습 | 빠른 경험 |

### 캠프의 구조

모든 캠프는 이런 구조를 따른다:

```
1. 랜딩 페이지 - 캠프 소개, 신청
2. 워크북 - Day별 콘텐츠 + 미션
3. 결과물 - 참가자가 만든 것
```

### 콜라보

캠프의 핵심 중 하나가 **콜라보**다.

- 다른 브랜드/크리에이터와 함께 진행
- 서로의 고객에게 노출
- 새로운 주제 실험

예: **크크웍스 × 실험마켓** = AI 포트폴리오 챌린지

## 3. 이 프로젝트의 위치

```
실험마켓 (브랜드)
├── tryoutmarket.co (메인 플랫폼)
│   ├── 수익화 코스
│   ├── 스킬 코스
│   └── 커뮤니티
│
└── tryout-camp (이 프로젝트)
    └── 캠프들
        └── AI 포트폴리오 등
```

캠프는 실험마켓의 **전초기지**다. 여기서 실험하고, 잘 되면 메인으로 확장한다.

## 4. 톤앤매너

캠프에서 사용하는 말투/분위기:

### DO (이렇게)
- 친근하게, 반말도 OK
- "해보자", "같이 하자"
- 실패해도 괜찮다는 느낌
- 구체적인 예시와 함께

### DON'T (이렇게 하지 말기)
- 딱딱한 강의체
- "~해야 합니다", "~하십시오"
- 완벽해야 한다는 압박
- 추상적인 설명만

### 예시

❌ "포트폴리오는 자신의 역량을 효과적으로 전달하는 도구입니다."

✅ "포트폴리오? 그냥 '나 이런 거 할 수 있어'를 보여주는 거야. 완벽할 필요 없어."
