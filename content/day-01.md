# Day 1. AI 웹빌더 이해하기

> 🎯 오늘의 목표: AI 웹빌더가 뭔지 이해하고, 직접 만져보면서 "이걸로 진짜 사이트를 만들 수 있구나"를 체험하기.

---

## 왜 이걸 하나요?

"포트폴리오 만들어야지..."

이 생각, 몇 번이나 했나요?
그리고 매번 여기서 멈추죠.

- 코딩 몰라서
- 디자인 못해서
- 시간 없어서
- 뭐부터 해야 할지 몰라서

**다 핑계예요.** (미안해요, 솔직히 말할게요 😅)

2026년에는 이 핑계가 더 이상 안 통합니다. 왜냐하면 **AI가 다 해주거든요.**

"이런 사이트 만들어줘" 라고 말하면, AI가 진짜로 만들어줍니다. 
코딩 0, 디자인 0, 필요한 건 **내가 원하는 걸 말로 설명하는 능력**뿐이에요.

오늘은 이게 진짜인지 **직접 체험**하는 날입니다.

---

## Step 1: AI 웹빌더가 뭔지 알기 (5분)

### 1-1. AI 웹빌더란?

**프롬프트(말)로 웹사이트를 만들어주는 도구**예요.

"카페 소개 사이트 만들어줘" 라고 입력하면, AI가 알아서:
- 디자인을 잡고
- 코드를 짜고
- 바로 볼 수 있는 웹사이트를 생성해줍니다

30초면 됩니다. 진짜로.

### 1-2. 대표 도구 3가지

지금 가장 많이 쓰는 AI 웹빌더 3개예요. 전부 **무료**로 시작할 수 있어요.

| 도구 | 특징 | 난이도 | 무료 범위 |
|------|------|--------|-----------|
| **Lovable** (lovable.dev) | 가장 쉬움, 대화하듯 수정 | ⭐ 입문 | 무료 5개 프로젝트 |
| **v0** (v0.dev) | 디자인 퀄리티 높음, Vercel에서 만듦 | ⭐⭐ 중급 | 무료 크레딧 제공 |
| **bolt.new** | 자유도 높음, 풀스택 가능 | ⭐⭐ 중급 | 무료 티어 있음 |

> 💡 **이 챌린지에서는 Lovable을 메인으로 씁니다.**
> 가장 쉽고, 코딩 경험이 전혀 없어도 따라할 수 있어요.
> 물론 v0이나 bolt.new가 더 편하신 분은 그걸 쓰셔도 OK!

### 1-3. 기존 방식 vs AI 웹빌더

혹시 "예전에 시도해봤는데 어려웠어요" 하시는 분?

| | 기존 방식 | AI 웹빌더 |
|---|-----------|-----------|
| 코딩 | HTML/CSS/JS 직접 작성 | 프롬프트만 입력 |
| 디자인 | 피그마/포토샵 필요 | AI가 알아서 디자인 |
| 시간 | 최소 며칠~몇 주 | 30초~5분 |
| 수정 | 코드 찾아서 고치기 | "이 부분 바꿔줘" 한마디 |
| 배포 | 서버 세팅, 도메인 연결 | 버튼 하나 클릭 |

완전히 다른 세상이에요. 이제 직접 체험해봅시다.

## Step 2: Lovable 가입하고 둘러보기 (5분)

### 2-1. Lovable 접속

👉 [lovable.dev](https://lovable.dev) 에 접속하세요.

### 2-2. 회원가입

1. 우측 상단 **"Sign up"** 또는 **"Get started"** 클릭
2. **Google 계정으로 가입** (가장 빠름)
3. 가입 완료! 에디터 화면이 바로 뜰 거예요.

> 💡 **팁**: GitHub 계정이 있다면 GitHub으로 가입하세요.

### 2-3. 화면 둘러보기

가입하면 이런 화면이 보여요:

- **왼쪽**: 프롬프트 입력창 (여기에 "이런 거 만들어줘" 라고 씁니다)
- **오른쪽**: 미리보기 (AI가 만든 결과물이 여기에 나와요)
- **상단**: 프로젝트 이름, 설정 등

아직 아무것도 안 해도 돼요. 구경만 하세요.

## Step 3: 첫 사이트 만들어보기 (10분)

여기가 오늘의 하이라이트입니다. **AI한테 사이트를 만들어달라고 해볼 거예요.**

### 3-1. 프롬프트 입력하기

Lovable 프롬프트 입력창에 아래 문구를 **그대로** 복사해서 붙여넣으세요:

```
카페 소개 웹사이트를 만들어줘.

- 카페 이름: 모닝브루
- 분위기: 따뜻하고 아늑한 느낌
- 색상: 베이지 + 브라운
- 포함할 섹션:
  1. 메인 히어로 (카페 이름 + 한 줄 소개)
  2. 메뉴 소개 (커피, 디저트 3~4개)
  3. 카페 분위기 사진 영역
  4. 영업시간 + 위치 안내
  5. 푸터
```

Enter 누르고 기다리세요. **30초~1분** 정도 걸려요.

### 3-2. 결과물 확인하기

AI가 만든 사이트가 오른쪽에 나타납니다.

잠깐 구경해보세요.

**✏️ 솔직한 첫 반응을 적어보세요:**

> ⚠️ **"와, 이걸 AI가 만들었다고?"** — 대부분 이런 반응이에요.
> 30초 만에 디자인 + 코딩을 다 해준 거예요.
> 여러분이 한 건 **말로 설명한 것뿐**입니다.

### 3-3. 대화로 수정해보기

마음에 안 드는 부분이 있죠? AI한테 말하면 돼요.

아래 수정 요청을 **하나씩** 해보세요:

**수정 1:**
```
메뉴 섹션을 카드 형태로 바꿔줘. 각 메뉴에 가격도 넣어줘.
```

**수정 2:**
```
전체 배경색을 좀 더 따뜻한 크림색으로 바꿔줘.
```

**수정 3:**
```
모바일에서도 예쁘게 보이도록 반응형으로 만들어줘.
```

> 💡 **수정 팁**: 한 번에 여러 개 요청하면 AI가 헷갈릴 수 있어요.
> **하나씩** 요청하고, 결과 확인하고, 다시 요청하는 게 좋습니다.

**✏️ 수정 전후 차이가 있었나요? 어떤 점이 바뀌었나요?**

### 4-2. 프롬프트 작성하기

