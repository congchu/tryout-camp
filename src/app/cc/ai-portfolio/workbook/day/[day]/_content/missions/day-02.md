## Step 3: GitHub 포트폴리오 레포 구경하기 (10분)

### 3-1. 왜 GitHub 레포?

GitHub에는 개발자들이 만든 **오픈소스 포트폴리오**가 많아요.
디자이너, 개발자, 프리랜서들이 실제로 쓰는 포트폴리오 소스코드가 공개되어 있어요.

이걸 참고하면?
- Lovable보다 **훨씬 퀄리티 높은** 결과물
- 실제로 **프로로 보이는** 포트폴리오
- 클라이언트가 **신뢰할 수 있는** 사이트

코드를 직접 이해할 필요 없어요.
**AI한테 "이 레포처럼 만들어줘"** 라고 하면 됩니다.
