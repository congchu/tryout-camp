## Step 1: 크몽/숨고에 링크 넣기 (15분)

### 1-1. 왜 플랫폼에 넣어야 하나?

크몽, 숨고 같은 플랫폼에서 클라이언트가 **프리랜서를 고를 때** 뭘 볼까요?

1. 프로필 사진
2. 소개글
3. **포트폴리오**

포트폴리오 링크가 있으면 **신뢰도가 확 올라가요.**
"이 사람 진짜 하는 사람이구나"

### 1-2. 크몽에 넣기

**크몽 계정이 있다면:**

1. [크몽](https://kmong.com) 로그인
2. **마이페이지** → **프로필 관리**
3. 소개글에 포트폴리오 URL 추가

**소개글 예시:**
```
[포트폴리오 확인하기]
{내 포트폴리오 URL}

더 많은 작업물은 포트폴리오에서 확인해주세요!
```

✏️ 크몽에 넣었나요?
- [ ] 넣었어요
- [ ] 크몽 계정이 없어요 → 숨고로 넘어가세요
- [ ] 나중에 할게요

### 1-3. 숨고에 넣기

**숨고 계정이 있다면:**

1. [숨고](https://soomgo.com) 로그인
2. **고수 프로필** → **프로필 수정**
3. 소개 또는 포트폴리오 섹션에 URL 추가

✏️ 숨고에 넣었나요?
- [ ] 넣었어요
- [ ] 숨고 계정이 없어요
- [ ] 나중에 할게요

### 1-4. 다른 플랫폼들

프리랜서 플랫폼은 많아요. 쓰는 게 있으면 다 넣어두세요.

- [ ] 크몽
- [ ] 숨고
- [ ] 위시켓
- [ ] 프리모아
- [ ] 링크드인
- [ ] 기타: _______

### ✅ Step 1 체크리스트

- [ ] 크몽 또는 숨고 프로필에 URL 추가
- [ ] (선택) 다른 플랫폼에도 추가

## Step 2: 명함에 QR코드 넣기 (10분)

### 2-1. 왜 QR코드?

오프라인에서 사람 만났을 때,
"포트폴리오요? 잠시만요..." 하고 URL 불러주기 어색해요.

**QR코드** 있으면?
명함 건네면서 "스캔하시면 포트폴리오 바로 보여요" 끝.

### 2-2. QR코드 만들기

1. **[QR Code Generator](https://www.qr-code-generator.com/)** 접속
2. **URL** 선택
3. 포트폴리오 URL 입력
4. **Download** 클릭

✏️ QR코드 다운로드 했나요?
- [ ] 네!
- [ ] 나중에 할게요

### 2-3. 명함에 넣기

**명함이 있다면:**
- 기존 명함 디자인에 QR코드 추가
- 새로 만들 거면 미리캔버스, Canva 등 활용

**명함이 없다면:**
- 일단 QR코드만 저장해두세요
- 카톡 프로필에 넣어도 OK

✏️ 명함 업데이트 계획:
- [ ] 기존 명함에 QR 추가할 예정
- [ ] 새 명함 만들 예정
- [ ] 카톡 프로필에 넣을 예정
- [ ] 일단 저장만 해둘게요

### 2-4. 카카오톡 프로필에 넣기 (보너스)

카톡 프로필 링크에 포트폴리오 URL을 넣으면
누가 물어볼 때 "카톡 프로필 확인해보세요" 한마디로 끝나요.

1. 카카오톡 → 설정 → 프로필 관리
2. 링크 추가
3. 포트폴리오 URL 입력

✏️ 카톡에 넣었나요?
- [ ] 넣었어요
- [ ] 나중에 할게요

### ✅ Step 2 체크리스트

- [ ] QR코드 생성
- [ ] 명함 또는 카톡에 활용 계획 세우기

## Step 3: 클라이언트한테 보내는 멘트 (10분)

### 3-1. 언제 포트폴리오를 보내나?

- 플랫폼에서 문의가 왔을 때
- 지인이 외주 연결해줄 때
- 미팅 전에 먼저 보여줄 때
- SNS DM으로 문의 왔을 때

### 3-2. 멘트 템플릿 만들기

**상황 1: 플랫폼에서 문의 왔을 때**

```
안녕하세요! 문의 감사합니다.

말씀하신 {프로젝트 유형} 작업 가능합니다.
먼저 제 포트폴리오 확인 부탁드려요.

📎 {포트폴리오 URL}

확인 후 궁금한 점 있으시면 편하게 말씀해주세요!
```

**상황 2: 지인이 연결해줄 때**

```
안녕하세요, {지인 이름}님 소개로 연락드려요.
{하는 일} 전문으로 하고 있습니다.

포트폴리오 먼저 공유드릴게요!
📎 {포트폴리오 URL}

필요하신 작업 방향 말씀해주시면
일정이랑 견적 안내드리겠습니다.
```

**상황 3: 미팅 전 사전 공유**

```
안녕하세요, {미팅 날짜} 미팅 전에 포트폴리오 공유드립니다.

📎 {포트폴리오 URL}

미팅 때 뵙겠습니다!
```
