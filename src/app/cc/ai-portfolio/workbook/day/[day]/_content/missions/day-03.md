**고객을 설득하는 것**이에요.

누가 볼지 모르면, 뭘 보여줘야 할지도 모르겠죠?

## Step 4: 작업물 이미지 추가하기 (15분)

### 4-1. 왜 이미지가 중요한가?

클라이언트는 **글보다 이미지**를 먼저 봐요.
작업물 이미지가 없으면 신뢰가 안 생겨요.

### 4-2. 작업물 스크린샷 찍기

**스크린샷 찍는 법:**
- Mac: `Cmd + Shift + 4`
- Windows: `Win + Shift + S`

**좋은 스크린샷 기준:**
- 작업물 전체가 보이게
- 깔끔한 배경
- 적당한 해상도 (1200px 이상 권장)

✏️ 스크린샷 찍은 작업물:
1.
2.
3.

### 4-3. 이미지 파일 저장

프로젝트 폴더의 `public` 폴더에 이미지를 넣으세요.

```
my-portfolio/
├── public/
│   ├── project1.png
│   ├── project2.png
│   └── project3.png
```

### 4-4. AI한테 이미지 연결 요청

```
프로젝트 1 이미지를 /project1.png로 바꿔줘
프로젝트 2 이미지를 /project2.png로 바꿔줘
프로젝트 3 이미지를 /project3.png로 바꿔줘
```

### 4-5. 확인

✏️ 이미지가 잘 나오나요?
- [ ] 네, 다 잘 나와요
- [ ] 일부만 나와요
- [ ] 안 나와요 → 슬랙에 질문
