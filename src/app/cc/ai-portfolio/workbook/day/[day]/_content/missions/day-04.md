# Day 4. 배포해서 URL 받기

> **오늘의 목표**: 포트폴리오를 Netlify로 배포하고, 내 URL 받기

---

## 오늘 할 것

지금까지 만든 포트폴리오는 **내 컴퓨터에서만** 보여요.
오늘은 **세상에 공개**할 거예요.

1. Netlify로 무료 배포
2. 내 포트폴리오 URL 받기
3. 모바일에서 확인
4. Contact 작동 확인

**오늘이 끝나면 클라이언트한테 링크를 보낼 수 있어요!**

---

## Step 1: 빌드하기 (5분)

### 1-1. 빌드가 뭔가요?

"빌드"는 코드를 **배포용 파일**로 변환하는 거예요.
개발용 코드 → 실제 서비스용 파일

### 1-2. 빌드 실행

터미널에서:

```bash
cd ~/Documents/my-portfolio
npm run build
```

기다리세요. (1~3분)

### 1-3. 빌드 결과 확인

성공하면 `dist` 또는 `out` 또는 `.next` 폴더가 생겨요.

✏️ 어떤 폴더가 생겼나요?
- [ ] `dist`
- [ ] `out`
- [ ] `.next`
- [ ] 에러가 났어요 → 슬랙에 질문

### ✅ Step 1 체크리스트

- [ ] `npm run build` 성공
- [ ] 빌드 폴더 확인

## Step 2: Netlify 배포하기 (15분)

### 2-1. Netlify Drop 접속

**[app.netlify.com/drop](https://app.netlify.com/drop)** 에 접속하세요.

> **Netlify Drop**: 폴더를 드래그앤드롭만 하면 배포되는 서비스예요.
> GitHub 필요 없어요!

### 2-2. 빌드 폴더 업로드

1. 빌드된 폴더(`dist`, `out`, 또는 `build`)를 찾으세요
2. 해당 폴더를 Netlify Drop 페이지로 **드래그앤드롭**
3. 업로드 완료까지 대기 (1~2분)

> **주의**: 폴더 전체를 드래그해야 해요. 파일 하나하나가 아니라요.

### 2-3. URL 확인

업로드가 완료되면 URL이 나와요.

예: `https://random-name-12345.netlify.app`

✏️ 내 포트폴리오 URL:


### 2-4. 사이트 이름 변경 (선택)

기본 URL이 `random-name-12345` 같은 이상한 이름이에요.
바꾸고 싶으면:

1. Netlify에 회원가입/로그인
2. Site settings → Change site name
3. 원하는 이름으로 변경 (예: `jisu-portfolio`)

✏️ 변경한 URL (선택):

- [ ] 빌드 폴더 드래그앤드롭
- [ ] URL 받기
- [ ] (선택) 사이트 이름 변경

- [ ] 이미지가 다 보이나요?

✏️ 문제가 있다면:


### 3-2. 모바일에서 확인 (중요!)

URL을 **휴대폰**으로 열어보세요.
카톡으로 링크 보내서 열어봐도 돼요.

클라이언트가 폰으로 볼 수도 있으니까 **모바일 확인 필수**예요.

- [ ] 모바일에서 잘 열리나요?
- [ ] 글씨가 너무 작지 않나요?
- [ ] 버튼 누르기 쉬운가요?
- [ ] 이미지가 잘 보이나요?

### 3-3. 모바일 문제 수정 (필요시)

문제가 있으면 AI한테 수정 요청하세요:

```

```

수정 후:
1. `npm run build` 다시 실행
2. Netlify Drop에 다시 업로드

## Step 4: Contact 확인하기 (10분)

### 4-1. 왜 Contact가 중요한가?

클라이언트가 포트폴리오 보고 연락하고 싶을 때,
**연락 방법이 없으면** 그냥 나가버려요.

Contact가 제대로 작동하는지 꼭 확인하세요.

### 4-2. Contact 섹션 체크

- [ ] 이메일 주소가 보이나요?
- [ ] 이메일 클릭하면 메일 앱이 열리나요?
- [ ] 다른 연락처(카톡, SNS)가 있나요?

### 4-3. 이메일 링크 확인

이메일이 `mailto:` 링크로 되어 있어야 클릭하면 메일 앱이 열려요.

직접 클릭해서 테스트해보세요.

✏️ 이메일 링크가 작동하나요?
- [ ] 네, 메일 앱이 열려요
- [ ] 아니요, 그냥 텍스트예요 → AI한테 수정 요청

**수정 요청 예시:**
```
이메일을 클릭하면 메일 앱이 열리게 mailto 링크로 바꿔줘
```

### 4-4. 카카오톡 오픈채팅 추가 (선택)

클라이언트가 카톡으로 연락하고 싶을 수도 있어요.
카카오톡 오픈채팅 링크를 추가하면 좋아요.

```
Contact 섹션에 카카오톡 오픈채팅 링크 추가해줘: {오픈채팅 URL}
```

### ✅ Step 4 체크리스트

- [ ] 이메일 주소 확인
- [ ] 이메일 mailto 링크 작동 확인
- [ ] (선택) 카카오톡 오픈채팅 추가
